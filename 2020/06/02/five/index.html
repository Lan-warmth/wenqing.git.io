下面例句一些常用例句：

创建数据库、数据表
– 创建数据库
create database 库名 charset=utf8;

– 使用数据库（使用后 才能进行创建表等操作）
use 库名;

– 创建表
create table 表名(
id int unsigned primary key auto_increment not null,
name varchar(20) default ‘’,
age tinyint unsigned default 0,
height decimal(5,2),
gender enum(‘男’,’女’,’中性’,’保密’) default ‘保密’,
cls_id int unsigned default 0,
is_delete bit default 0
);

说明：
name 称之为列名 ，varchar() 称之为列类型,以次类推
primary key：主键
auto_increment 自增长（1,2,3…）
not null：非空
unique: 唯一约束性
default ‘’ :默认为
decimal:小数，5为总长度，2为小数点后保留几位
enum：枚举类型

向表中插入数据
一条：insert into students values(1,’小明’,18,180.00,2,1,0)；
多条：insert into students values(1,’小明’,18,180.00,2,1,0),(0,’小明’,18,180.00,2,1,0);
tips：中文，日期用引号包起来,id只用填入第一个数据，后面的填0，会自动增长

查询所有字段

select * from 表名;

查询指定字段

select 列名1,列名2,… from 表名;

条件
使用where子句对表中的数据筛选，结果为true的行会出现在结果集中
例：select * from 表名 where 条件;
select * from students where id=1;

比较运算符
等于: =
大于: >
大于等于: >=
小于: <
小于等于: <=
不等于: != 或 <>

例：查询编号大于3的学生
select * from students where id > 3;

例：查询姓名不是“黄蓉”的学生
select * from students where id <= 4;

逻辑运算符
and
or
not

例：查询编号大于3的女同学
select * from students where id > 3 and gender=0;
例：查询编号小于4或没被删除的学生
select * from students where id < 4 or is_delete=0;

模糊查询
like
%表示任意多个任意字符
_表示一个任意字符

例：查询姓兰的学生
select * from students where name like ‘兰%’;
例8：查询姓兰并且“名”是一个字的学生
select * from students where name like ‘兰_’;
例9：查询姓兰或叫宁的学生
select * from students where name like ‘兰%’ or name like ‘%宁’;
